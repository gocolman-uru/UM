¿Como Buscar?

Se busca mediante un sistema de Labels (Los labels son palabras especificas que se encuentran en la base de datos que usamos para la Biblioteca digital) que permiten encontrar el contenido que uno esta buscando

Por ejemplo:

Si uno escribe Matematica, el searcher buscaria todos los libros disponibles que hablan sobre Matematica. Mientras mas labels especifcando el contenido que uno quiere encontrar, mas especifico seria el resultado que a uno le aparezca.

Ejemplos de LABELS

Autor, Editorial, Titulo, Materia, Temario especifico, Año de publicacion, Idioma
