¿Como usar el Scanner de Apuntes?

A la hora de usar el scanner, vas a tener que subir una imagen de tus resumenes. Nuestro modelo de Machine Learning detectará las palabras claves (Labels) que esten escritas, y luego te recomendaremos aquellos libros que concuerden con los Labels mapeados

¿Que son los labels y como funciona la busqueda de libros?

Cuando hablamos de Labels, estamos hablando de una serie de palabras o simbolos claves que nosotros creamos de acuerdo a la bibliografia indicada, permitiendo que cuando uno suba una foto a la pagina, algortimos de OCR detecten el texto y recomiende los libros que compartan dichas palabras o simbolos claves, tambien nos permite que la busqueda de dichos libros en la pagina web sea mucho mas simple mediante estos labels.
